# Dữ liệu mẫu

Thư mục này chứa dữ liệu mẫu cho các collection giống như trong ảnh chụp MongoDB Compass:

- `categories.json`
- `products.json`
- `users.json`
- `orders.json`
- `contactmessages.json`
- `subscriptions.json`

Mỗi file là một mảng JSON dùng để tham khảo hoặc nhập vào MongoDB.

Gợi ý sử dụng:
- Với MongoDB Compass: import từng file vào collection tương ứng.
- Với script/seed: đọc file JSON và tạo bản ghi theo mô hình hiện tại.
